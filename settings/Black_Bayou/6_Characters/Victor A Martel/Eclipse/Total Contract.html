Eclipse Security Contract #1 (Jameson Inns) ~ Business

2) Players involved (screen and character names)

Victor Martel (Victor A Martel)
Dashel Jameson (Dashel Jameson)

3) Summary (what you want to happen)

This is a standardized Employment Contract that will allow monitoring of sights and activities as per the Eclipse SL. This includes motion sensors, video cameras and all standard gear. This contract is for a six month time period which will have to be resubmitted should any change in contract be made prior to then or the limit of this contract is ceased.

Security Features Offered

1) Physical monitoring by security guards that have been hired, podunk security guards are merely low rank NPC's that are simply there to call the police in case of breakins.

2) Through use of cameras supplied by Fairlights old security branch, which Eclipse took over, we monitor and record all cameras used. All buildings protected by Eclipse have cameras, which are passed to Eclipses Main office and Fairlight Main CPU. The controls for this monitoring is done only at Victor's office and to access this system you must enter ten (10) sixteen (16) digit alpha-numeric passwords only known to Victor.

3) Standard Security. Motion sensors, laser sensors, pressure and glass sensors, door sensors and thermal sensors. The thermal sensors will go off if there is any sudden ten degree difference anywhere in the room. The video feeds report the feed levels to the systems and monitors so that any attempts to patch feeds from them will register as a decrease in feed. This works for all security devices.

4) All features route to Eclipse and the police can be called with the touch of a button reporting the adress direct to dispatch.

5) All Eclipse security is satelite uplinked and may be accessed through the SatLink that is in Victors office at home.

Standard Equipment

All Eclipse Security personnel get the following equipment.

Security Badge (Low End guards get standard not the Quick-Rip ones)
Jacket with Eclipse Security logo
Black Battle Dress Uniform
High quality steel toed black boots.
Sidearm, registered with Local and Federal officials
Stun Baton, also registered.
