Sending Host and ST Logs
Each Host Log is to be sent to BBST Isil, BBST Martel, and BBSTMail immediately after the end of the shift. For the Subject of the email use the following format:
(Name) Host Log (Date) (Shift) i.e. Deceit Host Log 10-19-98 6-8

The title of the Log file must be in this format:
(Date-mmddyy)s(Number) i.e. 101998s1 (translation: 10-19-98 shift 1)
s1 = 6-8, s2 = 8-10, s3 = 10-12, s4 = 12-2

Each ST Log is to be sent to BBST Isil and BBSTMartel immediately after the ST session ends. For the Subject of the e-mail, each ST Log shall have the following format:
(Name) ST Log (Date) i.e. Deceit ST Log 10-19-98
The title of the Log file must be in this format:
ST (Date-mmddyy) (Name) i.e. ST 101998 Deceit

In the body of each ST session email, there is to be specific information regarding the ST session. The items that must be noted: 1) name of ST, 2) date of the ST session, 3) names of the players involved, and 4) a brief summary of the ST session, including important information such as if a character has died. As soon as we finish the ST forms, a copy will be sent to you and you will be expected to use it.
